MathExpansion::Matriz_Densa

Esta clase permite representar matrices densas. Las matrices densas son aquellas que no tienen más de un 60% de elementos nulos. Su representación será muy similar a como se suelen representar tradicionalmente.

def self.leerArray(array)
  raise ArgumentError , 'Tipo invalido' unless array.is_a? Array
  
  c = Matriz_Densa.new(array.size(), array[0].size())
  array.each_index do |i|
    array[i].each_index do |j|
      c.set(i, j, array[i][j])
    end
  end
  c
end

def initialize(n, m)
  super
  
  @contenido = Array.new(@N,0)
  i = 0
  while i < @N
    @contenido[i] = Array.new(@M,0)
    i += 1  
  end
end

# File lib/math_expansion/matriz_densa.rb, line 38
def null_percent
   total = @N*@M
   no_nulos = 0
   
   i = 0
   while(i < @N)
     j = 0
     while(j < @M)
       if(@contenido[i][j] != @contenido[i][j].class.null)
         no_nulos += 1
       end
       j += 1
     end
     i += 1
   end
   
   nulos = total - no_nulos
   nulos.to_f/total.to_f
 end

to_s() click to toggle source

Permite representar de manera gráfica la matriz por consola.

  • Devuelve    :

    • Objeto del tipo string con todos los elementos de la matriz.

# File lib/math_expansion/matriz_densa.rb, line 85
def to_s
  s = ""
  i = 0
  while(i < @N)
    j = 0
    while(j < @M)
      s += "#{@contenido[i][j].to_s}\t"
      j += 1
    end
    s += "\n"
    i += 1
  end
  s
end
